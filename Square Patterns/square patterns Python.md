# Can you code these Square patterns with python


### Pattern 1

input : 5
```
* * * * *
* * * * *
* * * * *
* * * * *
* * * * *
```

<details> <summary>Show Solution</summary>

Solution by : [@pyGuru](https://github.com/pyGuru123)

```python
n = 5
for i in range(n):
  for j in range(n):
    print('*', end=' ')
  print()
```
